Spotify Web Playback SDK

This app uses the following:

  • Implicit Grant Flow with the Accounts API — to get an access_token to call the Web API & Web Playback SDK
  • Web Playback SDK (beta) — to create a Spotify Player inside of our web app (Spotify Premium required)
  • Transfer a User's Playback API (beta) — to transfer playback from Spotify's other clients to our Spotify Player
  • Start/Resume a User's Playback API (beta) — to initiate playback on our Spotify Player.
  • Get Audio Features — to obtain the danceability of the currently playing track.

If everything is set up properly, you should hear some music!
